# Ramanujan PI Series

This Go program calculates an approximation of π using the Ramanujan series.

##  Features

* High-precision π approximation
* Based on Ramanujan’s series

##  Usage

```bash
go run RamanujanPISeries.go
```

##  How It Works

The program implements Ramanujan’s formula for π, which converges extremely quickly compared to many classical series. Each additional term significantly increases accuracy.
